# Built-In math functions in Java

1. Power  - Invoked by Math.pow

```java
public class math {
    public static void main(String[] args) {
        double power = Math.pow(2,3);
        System.out.println(power);

    }
}
```

1. Pi Value - Math.PI
2. Euler’s value - Math.E
3. Absolute Value - Distance away from 0 - Math.abs
4. Square root - Math.sqrt
5. Round - Round to nearest whole integer - Math.round
6. Ceil - Always Round up - Math.ceil - Ex - 3.14 , it will round to 4
7. Floor - Always round down - Math.floor - Ex - 3.99 , It will round to 3.0
8. Max - max between two values - Math.max(num1, num2)
9. Min - same as above - Math.min(num1, num2)
